# Telecom AI Traffic Forecasting and Congestion Prediction

This project develops Machine Learning models to forecast telecom network traffic and predict cell congestion using historical telecom KPI data.

## Project Overview

Telecom operators need accurate traffic forecasting to optimize network resources and avoid congestion. This project uses machine learning techniques to predict traffic demand and identify potential congestion in advance.

## Features

- Traffic Forecasting using ML models
- Congestion Prediction based on PRB utilization
- Feature engineering with lag features and rolling averages
- Model comparison using multiple algorithms
- Feature importance analysis

## Machine Learning Models Used

- Linear Regression
- Random Forest
- XGBoost

## Technologies Used
